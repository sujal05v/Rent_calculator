## 🏠 Rent Calculator

A simple Python project built while learning programming concepts, designed to calculate how much each person needs to pay when sharing living expenses in a flat.

## 💡 Overview

This project demonstrates the use of basic Python concepts to solve a real-life problem — splitting shared expenses fairly among multiple people.

It takes inputs like rent, food expenses, and electricity usage, then calculates the per-person cost.

## ⚙️ Functionality

###  Accepts user inputs:

* Total rent

* Food/snack expenses

* Electricity consumption

* Cost per electricity unit

* Number of people

### Calculates:

* Total electricity bill

* Overall expense

* Equal share per person
